# NHIS_Project1: Bike Rental Demand Insights
This Project is about the data analysis of "Rental Bike Sharing" datasets. 
Dataset_1_YS: Is the first Dataset with 610 entries with 10 columns
Dataset_2_YS: Is the second Dataset with 610 entries with 8 columns
Dataset_3_YS: Is the third Dataset with 390 entries with 16 columns
Dataset_Merge_YS_v1.1: Is the Dataset created using merging Dataset 1 and Dataset 2, we get 610 entries with 16 columns now
Dataset_Final_YS_v1.2: Is the final Dataset created appending Dataset 3 with the Dataset_Merge, where we get 1000 entries with 16 columns initially.
But for final data analysis, we add several new columns and removed some, so the final number of columns will be 19
Project_One_YS: Is the final ppt which contains the insights about Analysis of Usage Patterns, Weather Impact and User Behaviour when come to the Rental Bike Demand.
